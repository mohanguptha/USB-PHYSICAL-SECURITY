# USB-PHYSICAL-SECURITY

The USB Physical Security System is a cybersecurity-focused desktop application designed to control and secure USB port access on a computer. Unauthorized use of USB devices is a common security risk, as it can lead to data theft, malware infections, and system compromise.

This project provides a simple yet effective solution by allowing users to enable or disable USB ports through a password-protected graphical interface. Only authorized users with the correct password can modify the USB port status, ensuring controlled access to external storage devices.

The application is built using Python and uses system-level batch scripts to enforce USB control policies. It integrates a user-friendly interface with essential security functionality, making it suitable for educational institutions, organizations, and personal systems.

This project demonstrates key cybersecurity concepts such as:

Endpoint security
Access control mechanisms
Protection against physical data exfiltration
Basic system hardening techniques

Overall, it serves as a practical implementation of physical security in cybersecurity, helping to prevent unauthorized data transfer and improve system protection.
